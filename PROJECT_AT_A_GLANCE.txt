================================================================================
                        QUICK BUG TRIAGE GUIDE
================================================================================

When a bug is reported, ask:
  1. Can the player play the game? NO ‚Üí CRITICAL
  2. Does a major feature work?     NO ‚Üí HIGH
  3. Is it cosmetic?                YES ‚Üí MEDIUM/LOW

CRITICAL (FIX NOW):
  CRITICAL ‚Üí Pull dev team, fix in <4 hours
  Example: Game won't start, 5-in-a-row broken

HIGH (FIX TODAY):
  HIGH ‚Üí Assign dev team, fix in <24 hours
  Example: One game mode broken, UI doesn't respond

MEDIUM (FIX THIS WEEK):
  MEDIUM ‚Üí Queue for dev team, fix in <2 days
  Example: Button styling wrong, minor rule issue

LOW (FIX IF TIME):
  LOW ‚Üí Backlog, fix before Dec 15 if possible
  Example: Typo, rare edge case, polish item
