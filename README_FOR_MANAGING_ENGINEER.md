**HIGH** ‚Üí Major feature broken but workaround exists  
‚Üí Fix ASAP (<24 hours), assign dev team today

**MEDIUM** ‚Üí Feature degraded, cosmetic issue  
‚Üí Fix this week (<2 days), queue for dev team

‚Üí Fix if time permits, can defer to post-release
